# MAS-MCloudBackup

用莫盘备份你的MAS存档


> ⛔注意：本模组已经停止对MAS版本<0.12.10进行维护，旧版本最后支持的版本为`1.6.1`

----------------

## 使用教程

1. 登录莫盘，进入webdav界面，点击 **创建新账号**
2. 备注名随便写，相对目录建议留空  
3. 用记事本打开 **game/Submods/MCloudBackup/MCloudBackup.rpy**，修改账号密码即可
4. 每次启动游戏都会进行备份
